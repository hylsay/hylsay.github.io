插件介绍

WordPress文章阅读插件Hylsay-Text-Reading是由hylsay独立开发，它的特点就是能把文字合成语音播放出来，非常适合小说类网站及新闻资讯类网站等等。

后台预览图

插件设置

API设置

本插件是基于百度语音合成开发，需要自行申请百度语音合成API，地址：http://ai.baidu.com/tech/speech/，申请好之后，把对应的apiKey及secretKey填写到对应位置保存。

声音设置

插件内置有4种声音可以选择，分别是：标准女声，标准男声，情感女声，情感男声。大家可以根据个人喜好进行选择。

阅读范围设置

阅读范围指的是插件阅读的区域范围，系统默认的是article，通常情况下Wordpress的文章都在article标签内，可根据不同主题进行自定义设置。

阅读屏蔽设置

阅读屏蔽功能是为了帮助我们过滤一些不需要的信息，比如文章发布时间、作者、打赏模块、或者文章底部的版权信息等。

切记，如有多个标签需要过滤，一定要使用英文半角分割开。

插件下载地址

GitHub https://github.com/hylsay/Hylsay-Text-Reading

Wordpress官网 https://wordpress.org/plugins/hylsay-text-reading/

问题反馈

网友：配置完成之后，没有提示报错，但是点击阅读没有声音。

答：有可能是API申请完成之后，语音合成没有激活造成的，激活方法很简单，领取免费额度哪里点击立即领取就可以了哦。
